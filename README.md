==========

An implementation of the [Silhouette] persistence layer using [ReactiveMongo].

## Usage

In your project/Build.scala:

libraryDependencies ++= Seq(
  "com.mohiva" %% "play-silhouette-persistence-reactivemongo" % "VERSION"
)
